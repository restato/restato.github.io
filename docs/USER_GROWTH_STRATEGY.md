## 1. SEO 최적화

### 1.1 롱테일 키워드 타겟팅
각 도구별로 사용자들이 실제로 검색하는 롱테일 키워드를 타겟팅합니다.

| 도구 | 타겟 키워드 (한국어) | 타겟 키워드 (영어) |
|------|---------------------|-------------------|
| JSON 포매터 | "json 정리 사이트", "json 이쁘게" | "json beautifier online" |
| QR 코드 | "qr코드 만들기 무료", "qr코드 생성" | "free qr code generator" |
| Base64 | "base64 인코딩 사이트", "base64 디코딩" | "base64 encode decode online" |
| 색상 변환 | "hex rgb 변환", "컬러코드 변환기" | "hex to rgb converter" |
| 이미지 리사이저 | "이미지 크기 조절 무료", "사진 용량 줄이기" | "resize image online free" |
| UTM 빌더 | "utm 링크 생성기", "utm 파라미터 만들기" | "utm builder free" |

### 1.2 각 도구 페이지 SEO 강화
```astro
<!-- 각 도구 페이지에 추가할 요소들 -->

<!-- FAQ Schema 추가 -->
<script type="application/ld+json">
{
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",

      "acceptedAnswer": {
        "@type": "Answer",
        "text": "네, 완전히 무료이며 회원가입도 필요없습니다."
      }
    }
  ]
}
</script>

<!-- HowTo Schema 추가 -->
<script type="application/ld+json">
{
  "@type": "HowTo",
  "name": "QR 코드 만드는 방법",
  "step": [...]
}
</script>
```

### 1.3 URL 구조 최적화
현재: `/tools/json`
제안: 다국어 URL 별도 처리
- `/tools/json` (한국어 기본)
- `/en/tools/json` (영어)
- `/ja/tools/json` (일본어)

### 1.4 내부 링크 강화
- 관련 도구 간 상호 링크 추가
- 블로그 포스트에서 도구 링크 자연스럽게 삽입
- 홈페이지에서 인기 도구 섹션 추가

## 3. 소셜 미디어 전략

### 3.1 플랫폼별 전략

#### Twitter/X
- 개발자 타겟: 개발 팁과 함께 도구 공유
- 해시태그: #개발자도구 #webdev #프론트엔드

```
💡 JSON 데이터 정리하기 귀찮으셨죠?
무료 JSON 포매터로 한 번에 해결하세요!
👉 https://restato.github.io/tools/json

#개발자도구 #JSON #웹개발
```

#### LinkedIn
- 전문가 타겟: 생산성 향상 관점에서 접근
- PM, 마케터 대상 UTM, 텍스트 카운터 등 홍보

#### Instagram
- 디자이너 타겟: 색상 도구, 그라데이션 도구 시각적으로 공유
- 릴스: 도구 사용 데모 영상

### 3.2 정기 콘텐츠 시리즈
- **#오늘의도구**: 매일 하나씩 도구 소개
- **#개발팁**: 도구와 연계한 개발 팁
- **#디자인팁**: 색상/CSS 도구 활용법

