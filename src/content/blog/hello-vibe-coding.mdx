## Vibe Coding이란?

**Vibe Coding**은 AI와 함께 코딩하며 아이디어를 빠르게 구현하는 개발 방식입니다.

기존의 개발 방식과 다른 점:

- 🎯 **결과 중심**: 완벽한 코드보다 동작하는 것을 우선
- 🚀 **빠른 프로토타이핑**: 아이디어를 즉시 구현
- 🤝 **AI 협업**: Claude와 대화하며 문제 해결
- 📚 **학습 과정**: 만들면서 배우기

## 왜 Vibe Coding인가?

개발을 시작할 때 가장 큰 장벽은 "어디서부터 시작해야 할지 모르겠다"는 것입니다.

Vibe Coding은 이 문제를 해결합니다:

```
1. 아이디어가 떠오른다
2. Claude에게 설명한다
3. 함께 구현한다
4. 동작하는 것을 본다
5. 개선한다
```

## 이 블로그의 목표

이 블로그에서는:

1. **개발 일지** - 프로젝트 진행 과정을 기록
2. **삽질기** - 실패와 해결 과정 공유
3. **TIL** - 매일 배운 것들
4. **프로젝트 쇼케이스** - 완성된 프로젝트 소개
