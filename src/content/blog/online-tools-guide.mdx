## 생성기 (Generators)

### QR 코드 생성기
**URL이나 텍스트를 즉시 QR 코드로 변환**

- 마케팅 자료, 명함, 포스터에 바로 사용
- 크기 조절 및 PNG 다운로드 지원
- 회원가입 없이 무료 사용

[QR 코드 생성기 사용하기 →](/tools/qr-code)

### 비밀번호 생성기
**강력하고 안전한 비밀번호를 한 번의 클릭으로**

- 대문자, 소문자, 숫자, 특수문자 조합
- 8~64자 길이 설정 가능
- 비밀번호 강도 시각적 표시
- 클립보드 복사 지원

[비밀번호 생성기 사용하기 →](/tools/password)

### UUID 생성기
**개발에 필수적인 고유 식별자 생성**

- UUID v4 표준 지원
- 한 번에 여러 개 생성 가능
- 대소문자 및 하이픈 옵션
- API 개발, 데이터베이스 설계에 필수

[UUID 생성기 사용하기 →](/tools/uuid)

### Lorem Ipsum 생성기
**디자인 목업에 필요한 더미 텍스트**

- 문단, 문장, 단어 단위 생성
- 원하는 개수만큼 생성 가능
- 웹/앱 디자인 프로토타입에 필수

[Lorem Ipsum 생성기 사용하기 →](/tools/lorem-ipsum)

### 색상 팔레트 생성기
**조화로운 색상 조합을 자동으로**

- 보색, 유사색, 삼각배색 등 다양한 배색법
- 기본 색상 선택 후 자동 팔레트 생성
- 디자이너를 위한 필수 도구

[색상 팔레트 생성기 사용하기 →](/tools/color-palette)

### 해시 생성기

- MD5, SHA-1, SHA-256 지원
- 파일 무결성 검증에 유용
- 보안 개발에 필수

## 왜 이 도구들을 사용해야 할까요?

### 100% 브라우저 처리
모든 도구는 브라우저에서 로컬로 처리됩니다. **데이터가 서버로 전송되지 않아** 완벽한 개인정보 보호를 보장합니다.

### 회원가입 불필요
로그인이나 회원가입 없이 모든 기능을 **즉시 사용**할 수 있습니다.

### 완전 무료
모든 도구가 **무료**입니다. 숨겨진 비용이 없습니다.

### 모바일 지원
**반응형 디자인**으로 스마트폰에서도 편리하게 사용할 수 있습니다.

### 다국어 지원
한국어, 영어, 일본어를 지원합니다.
