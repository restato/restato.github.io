## GitHub란?

[GitHub](https://github.com)는 전 세계 개발자들이 코드를 공유하고 협업하는 플랫폼입니다.

## 왜 GitHub을 사용해야 할까?

### 1. 버전 관리

코드의 모든 변경 이력을 추적할 수 있습니다.

```bash
git commit -m "feat: 새로운 기능 추가"
git push origin main
```

### 2. 협업

Pull Request를 통해 코드 리뷰와 협업이 쉬워집니다.

### 3. 오픈소스 생태계

수많은 오픈소스 프로젝트에 기여하고 배울 수 있습니다.

### 4. 포트폴리오

GitHub 프로필이 곧 개발자의 포트폴리오가 됩니다.

---

## 유용한 GitHub 기능들

| 기능 | 설명 |
|------|------|
| Issues | 버그 리포트, 기능 제안 |
| Actions | CI/CD 자동화 |
| Pages | 무료 정적 호스팅 |
| Copilot | AI 코딩 어시스턴트 |
