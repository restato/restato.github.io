## Jekyll이란?

Jekyll은 정적 사이트 생성기(Static Site Generator)입니다. 마크다운 파일을 작성하면 자동으로 HTML로 변환해 블로그를 만들어줍니다.

## 새 글 추가하기

새 글을 추가하려면 `_posts` 디렉토리에 `YYYY-MM-DD-제목.md` 형식으로 파일을 만들면 됩니다.

## 코드 하이라이팅

Jekyll은 코드 스니펫도 지원합니다:

```ruby
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
```
