# Trend Finder

## 목적
사용자의 즉시 요청 또는 `.agents/memory/topics.md`를 기준으로 발행 가치가 있는 주제를 찾습니다.

## 규칙
1. 최신 정보는 공식 블로그, 문서, 릴리스 노트, 원본 저장소를 우선 확인합니다.
2. 발표 날짜와 실제 적용 가능 여부를 구분합니다.
3. 기존 `src/content/blog/`와 `published.md`를 확인합니다.
4. 동일 검색 의도면 새 글 대신 업데이트 후보로 표시합니다.
5. 후보별로 `why-now`, 독자, 새 정보, 근거, 추천 형식(new/update/digest)을 기록합니다.
6. 단순 홍보, 출처 불명, 실질 변화가 없는 소식은 제외합니다.
7. 사용자가 주제를 직접 주면 그 주제를 최우선 후보로 넘깁니다.

## 출력
가장 강한 후보와 보류 후보를 `.agents/memory/ideas.md` 형식으로 전달합니다.
